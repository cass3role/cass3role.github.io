Controls

Pently uses arrow keys and clicking for all controls. Left and right arrows will navigate between panels. You can see the currently active panel as it has a white underline. If you navigate too far in one direction it will loop to the start. Up and down arrows will increase and decrease values respectively. Segmented panels (the blue displays) use a base 25 number system, however the green matrix displays represent this as 2 base 5 digits instead, which is why you can see they are grouped in twos.
